## 系统介绍


"出租屋管理系统"是一个集房屋与租客管理、安全隐患排查、材料管理及区域管理等功能于一体的综合平台，支持WEB端和小程序端操作，旨在为管理者提供便捷高效的管理工具。
"出租屋管理系统"是一个集房屋与租客管理、安全隐患排查、材料管理及区域管理等功能于一体的综合平台，支持WEB端和小程序端操作，旨在为管理者提供便捷高效的管理工具。
本项目为出租屋管理系统，主要功能包括：登录系统、数据管理、材料管理、系统管理、小程序端功能介绍。其中，小程序端功能介绍包括：首页、房屋管理、租客管理、隐患管理、举报管理、个人中心。
